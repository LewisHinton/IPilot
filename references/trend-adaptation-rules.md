# Trend Adaptation Rules

IPilot supports trend-based and meme-native assets when the user wants topical promotion or social media momentum.

The goal is to adapt the IP to a trend without destroying IP identity, copying protected characters, or producing unsafe content.

## Supported trend use cases

- platform-native meme formats;
- seasonal social media trends;
- safe pop-culture-adjacent formats;
- non-political community jokes;
- brand-safe challenge formats;
- holiday and festival trend adaptation;
- safe remix of common internet expressions;
- technical-community jokes;
- campus-life jokes.

## Trend intake questions

Ask:

1. What trend, meme format, platform habit, or topic do you want to reference?
2. Which platform is this for?
3. Should the result be obvious, subtle, parody-like, or only inspired by the trend?
4. What must not be copied directly?
5. What safety strictness level should apply?
6. Should the IP remain fully recognizable, or can it wear temporary trend-themed props?

## Safe adaptation principle

Prefer adapting:

- structure;
- emotion;
- rhythm;
- composition logic;
- social context;
- caption pattern.

Avoid directly copying:

- copyrighted characters;
- distinctive artwork;
- exact slogans if protected or brand-associated;
- living artist style;
- hateful, harassing, or tragedy-related memes;
- political persuasion content;
- real-person mockery without permission.

## IP preservation rule

Even when using a trend, preserve:

- identity anchors;
- line language;
- drawing style;
- color ratios;
- signature accessory;
- shape construction;
- brand-safe tone.

The trend should become a costume, scene, caption logic, or composition layer. It should not replace the IP.
